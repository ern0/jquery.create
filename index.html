You can imagine how surprised I was when I found in 2013 summer that there is no jQuery module yet which wraps document.createElement()!

So, I've made it, here is the long-waited jQuery plugin called:

Usage:
 $("#container").append( $.create("div").addClass("box").html("Hello, world!") );
 $.create("tr","row-8"); // second parm: ID

I know, it's not the most useful plugin of the jQuery world, but I think $.create("div") looks better than elm = $("<div></div>") or elm = $("<div>"). And it's also free! And lightweight!
